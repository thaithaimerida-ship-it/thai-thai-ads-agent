## Task 2: `fetch_campaign_geo_criteria()` in `ads_client.py`

**Files:**
- Modify: `engine/ads_client.py` (add after `_remove_existing_geo_criteria`)

The function does TWO GAQL queries:
1. All ENABLED campaigns (id, name, advertising_channel_type)
2. All LOCATION criteria for ENABLED campaigns (campaign_id, geo_target_constant resource_name)

It cross-references them and returns a dict keyed by campaign_id string.

`geo_target_constant` resource name format: `geoTargetConstants/1010182` → extract last part as location_id.

- [ ] **Step 1: Write the function** (insert after `_remove_existing_geo_criteria` ~line 820)

```python
def fetch_campaign_geo_criteria(client, customer_id: str) -> dict:
    """
    Retorna un dict keyed by campaign_id con los criterios de geotargeting de
    todas las campañas ENABLED.

    Hace dos queries GAQL:
      1. Todas las campañas ENABLED (id, name, advertising_channel_type).
      2. Todos los criterios LOCATION positivos de campañas ENABLED.

    El resultado incluye tanto las campañas con ubicaciones configuradas como
    las que no tienen ninguna restricción geográfica explícita.

    Formato del dict retornado:
    {
      "123456": {
        "campaign_id":             "123456",
        "campaign_name":           "Thai Merida Local",
        "advertising_channel_type": "SEARCH",
        "location_ids":            ["1010182"],
        "criteria_resource_names": ["customers/.../campaignCriteria/..."],
      },
      ...
    }
    Campañas sin ningún criterio LOCATION aparecen con location_ids=[].
    """
    from config.agent_config import GEO_EXCLUDED_CHANNEL_TYPES
    ga_service = client.get_service("GoogleAdsService")

    # ── Query 1: todas las campañas ENABLED ──────────────────────────────────
    q_campaigns = """
        SELECT
          campaign.id,
          campaign.name,
          campaign.advertising_channel_type
        FROM campaign
        WHERE campaign.status = 'ENABLED'
    """
    result: dict = {}
    try:
        for row in ga_service.search(customer_id=customer_id, query=q_campaigns):
            cid = str(row.campaign.id)
            channel_type = row.campaign.advertising_channel_type.name
            result[cid] = {
                "campaign_id":              cid,
                "campaign_name":            row.campaign.name,
                "advertising_channel_type": channel_type,
                "location_ids":             [],
                "criteria_resource_names":  [],
            }
    except Exception as e:
        print(f"fetch_campaign_geo_criteria: error en query campañas — {e}")
        return {}

    # ── Query 2: criterios LOCATION positivos de campañas ENABLED ───────────
    q_criteria = """
        SELECT
          campaign.id,
          campaign_criterion.location.geo_target_constant,
          campaign_criterion.resource_name
        FROM campaign_criterion
        WHERE campaign.status = 'ENABLED'
          AND campaign_criterion.type = 'LOCATION'
          AND campaign_criterion.negative = FALSE
    """
    try:
        for row in ga_service.search(customer_id=customer_id, query=q_criteria):
            cid = str(row.campaign.id)
            if cid not in result:
                continue  # campaña no en result (seguridad)
            geo_rn = row.campaign_criterion.location.geo_target_constant
            # Extraer location_id del resource name "geoTargetConstants/1010182"
            location_id = geo_rn.split("/")[-1] if geo_rn else ""
            if location_id:
                result[cid]["location_ids"].append(location_id)
                result[cid]["criteria_resource_names"].append(
                    row.campaign_criterion.resource_name
                )
    except Exception as e:
        print(f"fetch_campaign_geo_criteria: error en query criterios — {e}")
        # No retornar vacío — devolver lo que tenemos de la query 1

    return result
```

```python
"""
Thai Thai Ads Agent — Detector de Geotargeting (Fase GEO MVP)

Señales implementadas:
  GEO1 — campaña tiene al menos una ubicación NO permitida.
  GEO0 — campaña no tiene ninguna restricción geográfica explícita.

Si una campaña tiene Mérida + otra ubicación no permitida → GEO1 (no GEO0).

Función pura — no realiza llamadas a la API de Google Ads ni a SQLite.
Testeable con datos sintéticos, sin dependencias externas.

Tipos excluidos (la API no permite mutar sus criterios geo de forma confiable):
  PERFORMANCE_MAX, SMART, LOCAL_SERVICES

Input esperado de `geo_criteria_by_campaign` (de fetch_campaign_geo_criteria()):
  dict keyed by campaign_id:
  {
    "campaign_id":              str,
    "campaign_name":            str,
    "advertising_channel_type": str,   # "SEARCH", "PERFORMANCE_MAX", etc.
    "location_ids":             list,  # ["1010182", ...] — vacío si sin restricción
    "criteria_resource_names":  list,
  }
"""

from config.agent_config import GEO_EXCLUDED_CHANNEL_TYPES


def detect_geo_issues(
    geo_criteria_by_campaign: dict,
    allowed_location_ids: set,
) -> list:
    """
    Detecta señales GEO1 y GEO0 en campañas activas.

    Solo evalúa campañas elegibles (excluye PERFORMANCE_MAX, SMART, LOCAL_SERVICES).
    Solo propone (RISK_PROPOSE) — nunca ejecuta automáticamente.

    Args:
        geo_criteria_by_campaign : dict de fetch_campaign_geo_criteria()
        allowed_location_ids     : set de location_ids permitidos (ej. {"1010182"})

    Cada candidato retornado incluye:
      signal                  — 'GEO1' o 'GEO0'
      campaign_id             — ID de la campaña
      campaign_name           — nombre de la campaña
      advertising_channel_type — tipo de canal (para trazabilidad)
      detected_location_ids   — lista de IDs encontrados (vacía para GEO0)
      disallowed_location_ids — IDs no permitidos detectados (vacía para GEO0)
      allowed_location_ids    — set de IDs permitidos (para trazabilidad)
      severity                — 'high' para GEO1, 'low' para GEO0
      reason                  — texto descriptivo con evidencia

    Returns:
      Lista de dicts. GEO1 primero (mayor severidad), luego GEO0.
    """
    geo1_candidates = []
    geo0_candidates = []

    for cid, entry in geo_criteria_by_campaign.items():
        channel_type  = entry.get("advertising_channel_type", "")
        campaign_name = entry.get("campaign_name", "")
        location_ids  = entry.get("location_ids", [])

        # Excluir tipos no auditables
        if channel_type in GEO_EXCLUDED_CHANNEL_TYPES:
            continue

        disallowed = [lid for lid in location_ids if lid not in allowed_location_ids]

        if disallowed:
            # GEO1: al menos una ubicación no permitida
            geo1_candidates.append({
                "signal":                   "GEO1",
                "campaign_id":              cid,
                "campaign_name":            campaign_name,
                "advertising_channel_type": channel_type,
                "detected_location_ids":    list(location_ids),
                "disallowed_location_ids":  disallowed,
                "allowed_location_ids":     list(allowed_location_ids),
                "severity":                 "high",
                "reason": (
                    f"Campaña '{campaign_name}' tiene ubicaciones no permitidas: "
                    f"{disallowed}. "
                    f"Permitidas: {sorted(allowed_location_ids)}."
                ),
            })
        elif not location_ids:
            # GEO0: sin restricción geográfica explícita
            geo0_candidates.append({
                "signal":                   "GEO0",
                "campaign_id":              cid,
                "campaign_name":            campaign_name,
                "advertising_channel_type": channel_type,
                "detected_location_ids":    [],
                "disallowed_location_ids":  [],
                "allowed_location_ids":     list(allowed_location_ids),
                "severity":                 "low",
                "reason": (
                    f"Campaña '{campaign_name}' no tiene restricción geográfica explícita. "
                    f"Sin ubicación configurada, la campaña puede mostrarse en cualquier lugar."
                ),
            })
        # else: todos los location_ids son permitidos → sin señal

    return geo1_candidates + geo0_candidates
```

# ── GEO ALERT EMAIL (Fase GEO) ────────────────────────────────────────────

def _build_geo_alert_block(alert: dict, index: int, total: int, base_url: str) -> str:
    """Bloque de texto para una señal geo individual."""
    token         = alert.get("approval_token", "")
    campaign_name = alert.get("campaign_name", "")
    campaign_id   = alert.get("campaign_id", "")
    signal        = alert.get("signal", "")
    reason        = alert.get("reason", "")
    disallowed    = alert.get("disallowed_location_ids", [])
    detected      = alert.get("detected_location_ids", [])
    allowed       = alert.get("allowed_location_ids", [])

    lines = [
        f"{'─' * 62}",
        f"SEÑAL {index} / {total}  [{signal}]",
        f"{'─' * 62}",
        "",
        f"  Campaña    : \"{campaign_name}\"",
        f"  ID         : {campaign_id}",
        f"  Señal      : {signal} — {'ubicación incorrecta' if signal == 'GEO1' else 'sin restricción geográfica'}",
        "",
        "  EVIDENCIA",
    ]

    if signal == "GEO1":
        lines += [
            f"    Ubicaciones detectadas   : {detected}",
            f"    Ubicaciones NO permitidas: {disallowed}",
            f"    Ubicaciones permitidas   : {allowed}",
        ]
    else:
        lines += [
            "    La campaña no tiene ningún criterio de ubicación configurado.",
            f"    Ubicaciones permitidas: {allowed}",
        ]

    lines += [
        "",
        "  RAZÓN",
        f"    {reason}",
        "",
    ]

    if signal == "GEO1" and token:
        approve_url = f"{base_url}/approve?d={token}&action=approve"
        reject_url  = f"{base_url}/approve?d={token}&action=reject"
        lines += [
            "  ACCIÓN PROPUESTA",
            "    Corregir geotargeting → dejar solo Mérida, Yucatán, México (1010182).",
            "    El agente eliminará las ubicaciones incorrectas y configurará solo Mérida.",
            "",
            "  RIESGO",
            "    Medio. Afecta el alcance de la campaña inmediatamente.",
            "    Reversible — puedes ajustar la ubicación en Google Ads en cualquier momento.",
            "",
            f"  APROBAR corrección:",
            f"      {approve_url}",
            "",
            f"  RECHAZAR (mantener como está):",
            f"      {reject_url}",
            "",
        ]
    else:
        lines += [
            "  ACCIÓN SUGERIDA",
            "    Revisar manualmente en Google Ads → Configuración → Ubicaciones.",
            "    Considera agregar Mérida, Yucatán, México (ID 1010182) como ubicación objetivo.",
            "",
        ]

    return "\n".join(lines)


def build_geo_alert_email_text(
    alerts: list, session_id: str, base_url: str
) -> tuple[str, str]:
    """Construye asunto y cuerpo del correo de alertas geo."""
    from config.agent_config import PROPOSAL_EXPIRY_HOURS

    geo1_count = sum(1 for a in alerts if a.get("signal") == "GEO1")
    geo0_count = sum(1 for a in alerts if a.get("signal") == "GEO0")
    n = len(alerts)

    parts = []
    if geo1_count:
        parts.append(f"{geo1_count} campaña(s) con ubicación incorrecta (GEO1)")
    if geo0_count:
        parts.append(f"{geo0_count} campaña(s) sin restricción geográfica (GEO0)")

    subject = f"[Thai Thai Ads] Geotargeting — {' · '.join(parts)}"

    expiry_utc   = datetime.now(timezone.utc) + timedelta(hours=PROPOSAL_EXPIRY_HOURS)
    expiry_str   = _to_merida(expiry_utc)
    timestamp_str = _to_merida(datetime.now(timezone.utc))

    header = "\n".join([
        "=" * 62,
        "  THAI THAI ADS AGENT — Auditoría de Geotargeting",
        f"  Ciclo : {session_id}",
        "=" * 62,
        "",
        f"  Detectado : {timestamp_str}",
        "",
        f"Se detectaron {n} campaña(s) con segmentación geográfica incorrecta o ausente.",
        "",
        "GEO1 (ubicación incorrecta): requiere corrección accionable.",
        "GEO0 (sin restricción): aviso informativo — revisar manualmente.",
        "",
        "El agente NO ejecutará ninguna corrección sin tu aprobación.",
        "",
        f"Las propuestas GEO1 expiran el {expiry_str}.",
        "",
    ])

    blocks = [
        _build_geo_alert_block(a, i + 1, n, base_url)
        for i, a in enumerate(alerts)
    ]

    footer = "\n".join([
        "=" * 62,
        "Thai Thai Ads Agent",
        "administracion@thaithaimerida.com.mx",
        "Generado automáticamente — no responder a este correo.",
        "=" * 62,
    ])

    body = header + "\n".join(blocks) + "\n" + footer
    return subject, body


def send_geo_alert_email(alerts: list, session_id: str) -> bool:
    """Envía el correo de alertas de geotargeting vía SMTP Gmail."""
    from config.agent_config import (
        EMAIL_SMTP_HOST, EMAIL_SMTP_PORT,
        EMAIL_FROM, EMAIL_FROM_NAME, EMAIL_TO,
        GMAIL_APP_PASSWORD, APPROVAL_BASE_URL,
    )

    if not alerts:
        return False
    if not GMAIL_APP_PASSWORD:
        logger.warning("send_geo_alert_email: GMAIL_APP_PASSWORD no configurado")
        return False

    subject, body = build_geo_alert_email_text(alerts, session_id, APPROVAL_BASE_URL)

    msg = MIMEMultipart("alternative")
    msg["Subject"] = subject
    msg["From"]    = f"{EMAIL_FROM_NAME} <{EMAIL_FROM}>"
    msg["To"]      = EMAIL_TO
    msg.attach(MIMEText(body, "plain", "utf-8"))

    try:
        with smtplib.SMTP(EMAIL_SMTP_HOST, EMAIL_SMTP_PORT) as smtp:
            smtp.ehlo()
            smtp.starttls()
            smtp.login(EMAIL_FROM, GMAIL_APP_PASSWORD)
            smtp.sendmail(EMAIL_FROM, [EMAIL_TO], msg.as_string())
        logger.info("send_geo_alert_email: correo enviado — %d alerta(s)", len(alerts))
        return True
    except Exception as exc:
        logger.error("send_geo_alert_email: error SMTP — %s", exc)
        return False
```

        # Detecta campañas con ubicaciones incorrectas (GEO1) o sin restricción
        # geográfica (GEO0). GEO1 genera correo accionable con link de aprobación.
        # GEO0 queda visible en el correo como aviso informativo.
        # ====================================================================
        geo_audit_result = []
        try:
            from config.agent_config import (
                GEO_AUDIT_ENABLED,
                DEFAULT_ALLOWED_LOCATION_IDS,
                GEO_ALERT_DEDUP_HOURS,
            )
            from engine.geo_auditor import detect_geo_issues as _detect_geo
            from engine.email_sender import send_geo_alert_email as _send_geo

            if GEO_AUDIT_ENABLED and engine:
                geo_criteria = engine["fetch_campaign_geo_criteria"](client, target_id)
                geo_candidates = _detect_geo(geo_criteria, DEFAULT_ALLOWED_LOCATION_IDS)

                geo1_to_propose = []
                for geo in geo_candidates:
                    geo_campaign_id   = geo["campaign_id"]
                    geo_campaign_name = geo["campaign_name"]
                    geo_signal        = geo["signal"]

                    # Clave de dedup: 'geo:{id}:{signal}'
                    geo_keyword = f"geo:{geo_campaign_id}:{geo_signal}"

                    if geo_signal == "GEO1":
                        if memory.has_pending_proposal(keyword=geo_keyword, campaign_id=geo_campaign_id):
                            # Ya existe propuesta activa → observar
                            memory.record_autonomous_decision(
                                action_type="geo_action",
                                risk_level=RISK_OBSERVE,
                                urgency="normal",
                                decision="observe",
                                campaign_id=geo_campaign_id,
                                campaign_name=geo_campaign_name,
                                keyword=geo_keyword,
                                evidence={
                                    "signal":       geo_signal,
                                    "reason":       geo["reason"],
                                    "block_reason": "pending_proposal",
                                    **{k: geo[k] for k in ("detected_location_ids", "disallowed_location_ids", "allowed_location_ids")},
                                },
                                session_id=session_id,
                            )
                        else:
                            # Nueva señal GEO1 → registrar propuesta con token
                            geo_token = secrets.token_urlsafe(16)
                            decision_id = memory.record_autonomous_decision(
                                action_type="geo_action",
                                risk_level=RISK_PROPOSE,
                                urgency="normal",
                                decision="proposed",
                                campaign_id=geo_campaign_id,
                                campaign_name=geo_campaign_name,
                                keyword=geo_keyword,
                                evidence={
                                    "signal":       geo_signal,
                                    "reason":       geo["reason"],
                                    **{k: geo[k] for k in (
                                        "detected_location_ids",
                                        "disallowed_location_ids",
                                        "allowed_location_ids",
                                        "advertising_channel_type",
                                    )},
                                    "dry_run": dry_run,
                                },
                                session_id=session_id,
                                approval_token=geo_token,
                                proposal_sent=False,
                            )
                            geo1_to_propose.append({
                                **geo,
                                "decision_id":    decision_id,
                                "approval_token": geo_token,
                            })
                    else:
                        # GEO0: solo registrar en DB (aviso informativo, sin token)
                        # Nota: dedup de GEO0 queda protegido por el check externo
                        # has_recent_alert("geo_alert", ...) que envuelve todo el email.
                        memory.record_autonomous_decision(
                                action_type="geo_action",
                                risk_level=RISK_OBSERVE,
                                urgency="normal",
                                decision="observe",
                                campaign_id=geo_campaign_id,
                                campaign_name=geo_campaign_name,
                                keyword=geo_keyword,
                                evidence={
                                    "signal":   geo_signal,
                                    "reason":   geo["reason"],
                                    **{k: geo[k] for k in ("detected_location_ids", "allowed_location_ids")},
                                },
                                session_id=session_id,
                            )

                # Enviar correo si hay GEO1 nuevas o GEO0
                geo0_list = [g for g in geo_candidates if g.get("signal") == "GEO0"]
                all_for_email = geo1_to_propose + geo0_list
                if all_for_email:
                    already_alerted = memory.has_recent_alert("geo_alert", GEO_ALERT_DEDUP_HOURS)
                    if not already_alerted:
                        geo_sent = _send_geo(all_for_email, session_id)
                        if geo_sent:
                            memory.mark_proposals_sent(
                                [p["decision_id"] for p in geo1_to_propose]
                            )
                            memory.record_autonomous_decision(
                                action_type="geo_alert",
                                risk_level=RISK_EXECUTE,
                                urgency="normal",
                                decision="alert_sent",
                                campaign_id="",
                                campaign_name="",
                                keyword="geo_alert",
                                evidence={"geo1_count": len(geo1_to_propose), "geo0_count": len(geo0_list)},
                                session_id=session_id,
                            )
                            logger.info(
                                "Fase GEO: correo enviado — GEO1=%d GEO0=%d",
                                len(geo1_to_propose), len(geo0_list),
                            )
                        else:
                            logger.warning("Fase GEO: correo no pudo enviarse")

                geo_audit_result = geo_candidates

        except Exception as _geo_exc:
            logger.warning("Fase GEO: error en geo_auditor — %s", _geo_exc)
            geo_audit_result = [{"error": str(_geo_exc)}]

        if geo_audit_result:
            results["geo_audit"] = geo_audit_result
```
